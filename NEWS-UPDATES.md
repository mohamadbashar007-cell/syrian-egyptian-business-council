# تحديث الأخبار بعد إطلاق الموقع

تم تجهيز الموقع ليقرأ الأخبار من الملف:

`public/content/news.json`

## طريقة التحديث السريعة

1. افتح ملف `public/content/news.json`.
2. أضف خبراً جديداً في أعلى القائمة بنفس الشكل:

```json
{
  "id": 7,
  "title": "عنوان الخبر",
  "date": "2026-06-23",
  "category": "أخبار المجلس",
  "summary": "ملخص قصير للخبر يظهر في البطاقات.",
  "image": "https://images.unsplash.com/photo-1556761175-4b46a572b786?q=80&w=1600&auto=format&fit=crop",
  "featured": true
}
```

3. استخدم واحدة من هذه الفئات فقط: `أخبار المجلس`، `فعاليات`، `بيانات صحفية`، `استثمار`.
4. ضع `featured: true` على خبر واحد فقط إذا أردته أن يظهر في مقدمة الصفحة الرئيسية.
5. ارفع الملف المحدث إلى الاستضافة. إذا كان النشر عبر GitHub/Netlify/Vercel، يكفي عمل commit للملف وسيعاد النشر تلقائياً.

## الخيار الأفضل للإنتاج

لإدارة الأخبار بدون تعديل ملفات يدوياً، اربط هذا الملف مع نظام إدارة محتوى مثل Decap CMS أو Sanity أو Strapi. البنية الحالية جاهزة لذلك لأن الواجهة تقرأ الأخبار من JSON مستقل، ويمكن لاحقاً استبدال مصدر `loadNews` باستدعاء API دون تغيير تصميم الصفحات.
